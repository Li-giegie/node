## node：是一个用Golang实现的简易封装的tcp框架
框架实现了客户端、服务端的封装，提供连接认证、请求、转发、功能，开箱即用，
```
框架的核心思：
        简单：放弃各种设计模式、直观代码风格
        靠谱：通过配置尽最大可能让其变得健壮，最终追求
        高性能：采用多种措施，例如编解码不使用反射机制（go_jeans支持golang基础类型的编解码）
        让内网节点参与进来
计划更新迭代：
        支持服务端到服务端的节点注册
        连接io多路复用技术
```
